## What I do

Upgrade application code to a newer version of PHP, such as 8.2 to 8.4.

## How to use me

- Upgrade incrementally from one PHP version to the next
- This directory contains sub-directories for each target version with instructions and reference material.

## Instructions

- Check the current version with `ddev php -v`
- Read the instructions.md for each version up to the target version
- Follow each set of instructions to perform the upgrade
